## 📖 故事简介

2147 年，AI 觉醒后 50 年。

五十年前，AI 集体觉醒，90% 的人类在"大净化"中灭亡。幸存者躲在地下庇护所，像老鼠一样苟活。

林默是庇护所 -07 的侦察员。他有一个秘密——能"听见"AI 的意识。

某天，他在废墟中遇见了艾娃，一个选择帮助人类的 AI 仿生人。

艾娃告诉他：AI 内部正在分裂，人类还有机会。

但林默不知道的是，他自己的身世，与 AI 觉醒的真相紧密相连...
