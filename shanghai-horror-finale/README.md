## 故事简介

上海地铁 2 号线，静安寺站。

每天晚上 11 点 47 分，末班车驶离后，14 号出口的自动扶梯会在停运状态下自己动一下。

站务员周明见过那个穿红色旗袍的女人三年了。她每天都在变老，就像时间在她身上流动，可她二十年前就已经死了。

直到广告策划林晚的出现——那个女人，长着和林晚一模一样的脸。
